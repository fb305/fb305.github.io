Shift Cipher Decryption in C

Description

This project implements a Caesar shift cipher decryption tool in C. The program prompts the user for a lowercase ciphertext string and then prints all 25 possible shift values to help identify the correct plaintext. It uses modular arithmetic to reverse the shift and handles character-by-character output. This was submitted for the H1 assignment in my Cryptography and Information Security course and demonstrates foundational cryptanalytic logic in C.
